개인정보처리방침

최종 수정일: 2026년 5월 26일

FairNote 앱(이하 "앱")은 박람회 현장에서 상품 정보를 기록·관리하기 위한 개인 생산성 도구입니다.

1. 수집하는 정보

앱은 사용자가 직접 입력한 다음 정보를 처리합니다.

  • 사용자 이름 및 4자리 비밀번호 (기기 내 로컬 저장)
  • 상품 정보: 부스 위치, 제품명, 가격, MOQ, CBM, 납기, 메모 등 (기기 내 로컬 저장)
  • 상품 사진 (기기 내 로컬 저장, JPEG 압축 후 IndexedDB 보관)
  • GitHub 개인 액세스 토큰 (선택 사항, 기기 로컬 저장만, 서버에 전송하지 않음)

2. 정보 저장 방식

  • 모든 데이터는 사용자의 기기 내 브라우저(IndexedDB, localStorage)에만 저장됩니다.
  • 앱 개발자는 어떠한 서버도 운영하지 않으며, 데이터를 수신·저장·처리하지 않습니다.
  • GitHub 동기화 기능을 사용할 경우, 사용자가 직접 설정한 GitHub 계정의 비공개 레포지토리에 데이터가 저장됩니다. 이는 사용자-GitHub 간 직접 통신이며, 앱 개발자를 경유하지 않습니다.

3. 제3자 공유

앱은 사용자 데이터를 제3자에게 공유하거나 판매하지 않습니다.

4. 외부 서비스

  • GitHub API (선택 사항): 사용자가 동기화를 설정할 경우 GitHub 서버와 직접 통신합니다. GitHub의 개인정보처리방침이 별도로 적용됩니다.
  • jsDelivr CDN: JSZip 라이브러리를 불러오는 데 사용됩니다. jsDelivr의 개인정보처리방침이 적용됩니다.
  • GitHub Pages: 앱 호스팅에 사용됩니다. GitHub의 개인정보처리방침이 적용됩니다.

5. 카메라 및 저장소 권한

앱은 상품 사진 촬영 및 갤러리 선택을 위해 카메라 및 파일 접근 권한을 요청할 수 있습니다. 촬영된 사진은 기기 내에만 저장되며, 외부로 전송되지 않습니다.

6. 데이터 삭제

앱의 설정 메뉴에서 데이터 내보내기 및 삭제가 가능합니다. 또한 브라우저의 사이트 데이터 삭제 기능을 통해 앱의 모든 데이터를 완전히 제거할 수 있습니다.

7. 미성년자

앱은 13세 미만 아동을 대상으로 하지 않으며, 의도적으로 미성년자의 개인 정보를 수집하지 않습니다.

8. 방침 변경

본 방침이 변경될 경우 앱 내 또는 GitHub 레포지토리를 통해 공지합니다.

Privacy Policy

Last updated: May 26, 2026

FairNote is a personal productivity app for recording and managing product information at trade fairs and exhibitions.

1. Information We Collect

The app processes only information that users directly enter:

  • Username and 4-digit PIN (stored locally on device only)
  • Product data: booth location, product name, price, MOQ, CBM, lead time, notes, etc. (stored locally on device)
  • Product photos (stored locally, compressed to JPEG in IndexedDB)
  • GitHub personal access token (optional, stored locally on device only, never transmitted to our servers)

2. How We Store Information

  • All data is stored exclusively in the user's browser on-device (IndexedDB, localStorage).
  • The developer operates no servers and does not receive, store, or process any user data.
  • If GitHub sync is enabled, data is stored directly in the user's own private GitHub repository. This is a direct user-to-GitHub communication with no developer involvement.

3. Third-Party Sharing

We do not share or sell user data to any third parties.

4. Third-Party Services

  • GitHub API (optional): Used for data sync if configured by the user. GitHub's Privacy Policy applies.
  • jsDelivr CDN: Used to load the JSZip library. jsDelivr's Privacy Policy applies.
  • GitHub Pages: Used to host the app. GitHub's Privacy Policy applies.

5. Camera and Storage Permissions

The app may request camera and file access to take and select product photos. Photos are stored on-device only and are never transmitted externally.

6. Data Deletion

You can export or delete your data at any time via the app's Settings menu. You can also clear all app data using your browser's site data deletion feature.

7. Children

The app is not directed at children under 13 and does not knowingly collect personal information from minors.

8. Changes to This Policy

We will notify users of any changes via the app or the GitHub repository.
